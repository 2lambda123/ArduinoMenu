template<typename I, typename... II>
class StaticMenu< I, II >

The StaticMenu class is a data composition to hold a set of data items this data enumeration can't be changed either in order or in size as its structure will NOT be stored on the final program (compile time structure) data items can however be dynamic or of mixed types
